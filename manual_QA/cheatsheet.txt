Загрузка Файлов. Для текущего проекта допустимые форматы - текст в формате csv и xls, xlsx		
Тип файла		
1.	текстовый	
2.	xls	
3.	xlsx	
4.	db	
5.	dll	
6.	exe	
7.	картинка	
8.	xml	
9.	html, css	
10.	архив	
11.	Серверные файлы (php, jsp, asp)	
12.	Скриптовые файлы (js, ... )	
Расширение файла		
1.	без расширения	
2.	стандартное расширение (txt, csv, xls, xlsx, ...)	
3.	картинка с расширением txt	
4.	текст с расширением png	
5.	очень длинное расширение	
6.	заканчивается спецсимволом (точка, запятая, точка с запятой и т.д.)	
7.	zip-архив, переименованный в xlsx	
Имя файла		
1.	1 символ	
2.	несколько символов	
3.	спецсимволы	
4.	очень длинное имя (255 символов и больше)	
5.	с пробелами	
6.	только пробелы	
	файл без имени	
7.	пробелы в начале имени	
8.	пробелы в конце имени	
9.	управляющие символы в имени	
10.	непечатные символы в имени	
11.	в имени только пробелы (ну и без расширения, конечно)	
12.	русские буквы	
13.	русские буквы в неверной кодировке	
14.	символы с Диакритическими знаками	
15.	имя файла в виде XSS, SQLInjection	
16.	начинается с точки	
Данные внутри csv		
1.	Пустой файл	
	Тип файла	
		csv(MS-DOS)
		csv(Macintosh)
		csv(Comma delimeted)
2.	Разные типы разделителей:	
2.1.		запятая
2.2.		точка с запятой
2.3.		табуляция
2.4.		вертикальная черта
3.	разделители внутри текста:	
3.1.		текст в кавычках
3.2.		текст в апострофах
4.	Число колонок:	
4.1.		одна
4.2.		несколько
4.3.		очень много (больше 255)
4.4.		неоднородное число колонок в разных строках
5.	Наполнение:	
5.1.		пустые колонки
5.2.		пустые строки
5.3.		пустой файл
5.4.		только разделители внутри файла (то есть все колонки пустые, но сами разделители есть)
5.5.		Часть данных пустая (и в строках и в колонках)
6.	Данные	
6.1.		Русский текст (все символы)
6.2.		Английский текст (все символы)
6.3.		Числа - отрицательные, положительные, нули, маленькие числа, большие числа, дробные числа
6.4.		Даты - даты сильно в прошлом, даты в будущем, даты сильно в будущем, разный формат дат, разные разделители, неверные даты.
6.5.		Все спецсимволы
6.6.		Управляющие символы
6.7.		NULL
6.8.		разные окончания строк (UNIX и DOS)
6.9.		непечатаемые символы
6.10.		многобайтные символы, например, знак евро
6.11.		\u0000 (nul-byte)
6.12.		Данные в виде SQL инструкций
6.13.		XSS
7.	Кодировки:	
7.1.		win-1251
7.2.		win-1252
7.3.		koi8-r
7.4.		koi8-u
7.5.		ANSII
7.6.		UTF-8
7.7.		UTF-8 without BOM
7.8.		OEM 855
7.9.		OEM 866
7.10.		Mac
		
Размер файла		
	0 byte	
	<1000 Kb	
	1000 Kb	
	1 Mb	
	>1 Mb	
	1 Gb	
	1 Tb	
	архив из 2tb, сжатый до допустимого размера	
		
		
		
Права:		
	    Защищенный xls	
	    Запароленный архив	
	    запароленный pdf	
	    Попытка загрузить файл без прав на его чтение	
		
		
		
Данные внутри xls и xslx:		
		
	    Данные на одном листе	
	    Данные не на первом листе	
	    данные на нескольких листах	
	    Вставлен рисунок	
	    вставлена диаграмма	
	    в ячейках есть формулы	
	    даты в разном формате	
	    Пустые файлы	
	    Размер больше положенного	
	    пустые строки сверху	
	    пустые колонки слева от данных	
	    несколько диапазонов данных на одном листе	
	    битый xls-файл (сохранить нормальный файл, потом открыть его в текстовом редакторе и испортить так, чтобы он перестал открываться)	
	    Null-байты	
	    управляющие и непечатные символы в данных	
	на одном из листов ссылки на другой лист	
	Формулы с ошибками (деление на ноль, отсутствующие значения, ссылки на другие листы, которых нет и т.п.)
